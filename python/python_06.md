## 해시 테이블(Hash Table)
: 해시 함수를 사용하여 변환한 값을 색인(index)으로 삼아 키(key)와 데이터(value)를 저장하는 자료구조
 -> 데이터를 효율적으로 저장하고 검색하기 위해 사용한다.

### 해싱 함수(암호화)
```
1 % 3 = 1 / 
2 % 3 = 2 /
3 % 3 = 0 / 
```
간단한 해싱 함수를 만들었는데, 위의 코드에서 0이 주소 가장 앞에 들어간다.   
-> 숫서가 보장되지 않는다.
```
a -> 97 % 3 = 1
b -> 98 % 3 = 2
c -> 99 % 3 = 0
d -> 102 % 3 = 0
```
**c, d 해싱 충돌**이 발생한다.
