## 기본 명령어

### git add
(local에 있는 파일을 stage에 추가한다)

### git commit
: stage에 올라간 파일에 쪽지를 적는다

### git push
: stage에 올라와있는 파일을 저장소로 옮긴다  
주의 : add, commit 없이 push 불가능
---
### git pull
(저장소는 있는데 업데이트 필요할 때 사용)
